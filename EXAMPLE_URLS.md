## üì• How to Download

### For Images:
1. Click the link above
2. Click "Download" or "Free Download" button
3. Choose the size you want (usually "Original" or "Large")
4. Save the file
5. Rename it to match the app's file names (e.g., `asd.jpg` for lion)

### For Sounds:
1. Click the Freesound link
2. Click "Download" button (you may need to create a free account)
3. Save the `.mp3` or `.wav` file
4. If it's `.wav`, convert to `.mp3` using Audacity or online converter
5. Rename it to match the app's file names (e.g., `lion.mp3`)

## üí° Tips for Finding Resources

3. **Filter by license** - Make sure it's "Free to use" or "CC0"
4. **Check image size** - Download at least 400x400px for good quality
5. **Test sounds first** - Download one sound, test it, then get more
